# Design DNA — EdgeSales by Edge of Company

## Visual Personality: dark-modern
## Layout: storytelling (immersive → narrative → proof → CTA)
## Typography: bold-display (Bebas Neue headlines / Barlow body)

### Color Palette
- Primary (Navy): #0F1E3C — `oklch(0.17 0.045 255)`
- Surface (Near-black): #0A1628 — `oklch(0.13 0.030 258)`
- Accent (Amber): #F59E0B — `oklch(0.78 0.17 75)`
- Accent-dim: #D97706 — `oklch(0.67 0.16 72)` (hover)
- Text Primary: #F8FAFC — near-white
- Text Muted: #94A3B8 — slate mid
- Card Surface: #111827 — `oklch(0.19 0.025 255)`

### Font Pairing
- Display: Bebas Neue — weight 400 (single weight, max impact at any size)
- Body: Barlow — weights 300 / 400 / 500 / 600 / 700

### Unique Design Choices
1. **Command-center hero** — asymmetric amber spotlight glow at upper-left (NOT centered), with diagonal grid-line overlay for a "war-room operations" feel. Massive headline uses extreme tracking with mixed amber/white treatment.
2. **Number-callout proof cards** — amber stat numbers with text-shadow glow, dark near-black cards with left amber border accent — feel like live dashboard tiles, not testimonial blocks.
3. **3-step horizontal timeline** — step numbers are oversized ghost text behind each card, creating depth without any images. An amber connecting line threads through the center.
4. **Full-bleed CTA gradient** — amber-to-navy diagonal, NOT purple-to-blue. Uses grain texture for distinctive finishing feel.
5. **Section rhythm** — alternating surface-50 / surface-100 sections with thin amber accent lines, giving the page a "circuit board" cadence.
